PROJETO

O que é o projeto?

O diversiFica é uma empresa fictícia feita para a finalização do Bootcamp de Desenvolvimento Web da Generation Brasil. Ela foi elaborada a partir do oitavo Objetivo de Desenvolvimento Sustentável da ONU, que propõe promover o trabalho decente e o crescimento econômico.

O projeto é um site de busca de vagas de emprego que sejam seguras, diversas e inclusivas, principalmente para o público LGBTQIA+ e/ou pessoas com deficiência(s).

Por que o projeto existe?

O diversiFica existe para trazer um local seguro para encontrar empregos mais justos, com diversidade, equidade e inclusão.

Segundo a ONU, até 2030, precisamos “Alcançar o emprego pleno e produtivo e trabalho decente para todas as mulheres e homens, inclusive para os jovens e as pessoas com deficiência, e remuneração igual para trabalho de igual valor”.

Mesmo com outras plataformas e sites de busca de emprego já existentes, acreditamos que há um mercado muito amplo para esta proposta.

Ferramentas Utilizadas para a Construção do Projeto

Metodologia ágil: SCRUM, Zoom e Bloco de Notas

Back-end: Java, MySQL, Spring Boot, Postman e Heroku

Front-end: HTML, CSS, Bootstrap, Angular e Netlify

Implementações futuras

Futuramente, gostaríamos de trazer uma acessibilidade maior para o site, com LIBRAS, navegação facilitada pelo teclado, leitor de texto, comando de voz, ajuste de cor, contraste e tamanho, entre outras melhorias.

Também desejamos transformar o site em aplicativo e ampliar suas funcionalidades, criando uma sessão para cursos com certificação, tanto para pessoas físicas, como para empresas, tendo, dessa forma, maior controle sobre a veracidade das informações fornecidas por nosses usuáries, e disseminando informações que nos auxiliam em sermos uma sociedade mais inclusiva.

Outras implementações também serão necessárias, como aumentar a segurança e melhorar a experiência do usuário.
